计算机网络知识点(七)：计算机网络体系结构（已迁移）

· 2021-12-18 · # 计算机网络

🤠 本章主要讲讲TCP/IP四层、五层协议、OSI七层协议体系结构

一、OSI七层模型

OSI七层协议模型主要是：

  1. 应用层
  2. 表示层
  3. 会话层
  4. 运输层
  5. 网络层
  6. 数据链路层
  7. 物理层

二、TCP/IP四层模型

TCP/IP是一个四层的体系结构，主要包括：

  1. 应用层
  2. 运输层
  3. 网际层
  4. 网络接口层

三、五层体系结构

五层体系结构包括：

  1. 应用层
  2. 运输层
  3. 网络层
  4. 数据链路层
  5. 物理层
